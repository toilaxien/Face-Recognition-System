🚀 AI-Powered Face Recognition & Automated Attendance System
Một hệ thống điểm danh thông minh tích hợp công nghệ nhận diện khuôn mặt tiên tiến nhất hiện nay (SOTA), được thiết kế để giải quyết bài toán quản lý nhân sự trong môi trường doanh nghiệp thực tế.

🌟 Key Highlights (Điểm nhấn dự án)
State-of-the-art Models: Triển khai đa dạng kiến trúc từ ArcFace + ResNet50 (độ chính xác cao) đến MobileFaceNet (tối ưu cho thiết bị di động/nhúng).

High Performance: Hỗ trợ huấn luyện đa GPU (Multi-GPU Training) và kỹ thuật Early Stopping để tối ưu hóa thời gian và hiệu suất hội tụ.

Industrial Standard: Quy trình xử lý hoàn chỉnh từ Face Detection -> Alignment -> Embedding Extraction -> Classification.

Benchmarking: Đánh giá nghiêm ngặt trên các tập dữ liệu tiêu chuẩn quốc tế: LFW, CFP-FP, AgeDB.
